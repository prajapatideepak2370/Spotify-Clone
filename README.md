# Spotify Clone

A simple static **Spotify Clone** UI built with HTML and CSS.

## Overview

This repository contains a static front-end clone of the Spotify interface. It includes `index.html`, `style.css`, and an `Assests/` folder with images used in the UI.

## How to view locally

- Open `index.html` in your browser, or use a simple server:
  - Python 3: `python -m http.server 8000`
  - VS Code: Open the folder and use Live Server extension

## Deployment

This project can be hosted with **GitHub Pages**. The site will be available at:

`https://prajapatideepak2370.github.io/Spotify-Clone/`

(Deployment may take a few minutes after enabling Pages.)
